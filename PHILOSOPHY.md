# Philosophy

Working philosophies particular to this project. Filled by the Setup Grill; grown only through the Retro Loop. The alignment agent enforces these plus the baked Principles.

## Project philosophies

### Src-First
Modify `src/` first, always. A change to how the framework works never starts in the root copies; update the root to match afterward.

### Precise Minimalism
Clear instructions, not wordy rules. Cut explanatory prose from shipped artifacts until only instruction remains.
