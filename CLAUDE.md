# CLAUDE.md

This project runs on the execution framework for autonomous, agent-driven work. Orient here first:

- `CONTEXT.md` — domain glossary and team roster.
- `PROCESS.md` — how work moves (the Lifecycle, the Retro Loop, QA gates). Invariant.
- `PHILOSOPHY.md` — working philosophies particular to this codebase.
- `TEAM_DIRECTIVE.md` — how the team interacts: roster, routing, gates, Don'ts.
- `.claude/agents/` — the team. Persistent Teammates and the Adherence Agents.
- `tmp/exec/` — sprint/issue/retro work-tracking (ephemeral, gitignored). Durable conclusions are promoted out into the docs above via the Retro Loop.

Start work with the Work Grill: `grill-with-docs` → `to-prd` → `to-issues`, in a session separate from setup.
